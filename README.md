# MARS Wildlife Rescue

This is a static version of the site that will be migrated to WordPress. It uses **Tailwind CSS** for styling.

## Getting Started

Because this project relies on Tailwind CSS, you must install the required dependencies and run a watch command before adding new code.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. In the Terminal paste: npm install tailwindcss @tailwindcss/cli
   This will add a node_modules folder
2. In the Terminal paste: npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
